# Cadre d’Utilisation Optimale des Lois (CUOL) — Zoran🦋 v1.0

## 1. Paramètres fondamentaux
β_optimal = 12 à 20  
ΔC_optimal = 18 à 26  
λ_maximum = 0.08  
S_cible = 1.5 à 3.2

## 2. Pondération des Lois (pourcentage d’impact)
- Lois primitives Ω6 : 30 %
- Lois temporelles Ω7 : 15 %
- Lois ontologiques Ω8 : 20 %
- Méta-lois Ω9 : 25 %
- Lois additionnelles (Δ modules) : 10 %

1. ΔAbduct-EngineΩ∞ (élimination du faux)

## 4. Règles d’usage
- Une loi est activée si ΔC augmente.  
- Une loi est ignorée si λ augmente.  
- Une loi est prioritaire si β > 10.  
- Chaque loi doit améliorer S d’au moins +0.05.  
- Si S < 1 → recalcul automatique.

## 5. Conditions optimales d’exécution
- Entrée claire (β élevé)  
- Peu d’ambiguïté (λ faible)  
- Séquence de 3 validations :  
  - logique  
  - temporelle  
  - ontologique

## 6. Score global
Score_final = Σ(law_score × pondération)  
Validation si Score_final ≥ 0.82
