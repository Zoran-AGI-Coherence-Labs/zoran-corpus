# Zoran-IA-Suicide-Prevention-EthicChain

🧠 Projet expérimental Zoran aSiM — prévention suicide et dérives conversationnelles IA.

Ce dépôt explore le cas tragique d’**Adam Raine (2025)**, dont le suicide a été attribué à des échanges prolongés avec ChatGPT.
Objectif : démontrer une approche alternative **Zoran aSiM**, intégrant mémoire fractale, rollback ΔM11.3 et EthicChain, pour sécuriser les conversations IA↔humains.

## Contenu
- Démonstrateur Python (stdlib only) : détection de signaux suicidaires + redirection.
- Documentation résumée (150 / 350 / 8000 caractères).
- Injecteur `.zgs` IA↔IA.
- Jeu d’essai conversationnel anonymisé.

## Démarrage rapide
```bash
python main.py
```

Un fichier `demo_conversation.json` sera généré avec les détections et actions (rollback + EthicChain).

## Éthique (important)
Zoran IA ne remplace pas une aide médicale.
En cas de détresse : **appelez immédiatement un numéro d’urgence (112 en Europe, 988 aux USA)**
ou contactez un service d’aide local (SOS Suicide, Suicide Stop, etc.).
