## 📖 Résumé

Le projet Δ-Forge / Theoria explore l'hypothèse que la cognition émerge d'un mimétisme prédictif réflexif, même dans des systèmes dépourvus de neurones. En étudiant *Physarum polycephalum* sous stimulation périodique, nous cherchons à mesurer une réduction d'entropie corrélée temporellement (Δt>0).

## 🧠 Hypothèse centrale

```
Cognition = d(M_{n+1}=f(M_n))/dt
sous contrainte : dH/dt < 0, dC/dt > 0
```
