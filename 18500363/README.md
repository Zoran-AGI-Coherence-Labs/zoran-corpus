# Structural Admissibility via Coherence Constraints

DOI: 10.5281/zenodo.18500363

## What this package is
A canonical, output-only admissibility framework for generative AI. The framework is **non-contournable** by:
- reformulation (synonyms / paraphrases),
- session reset,
- “clean” prompts that hide the same structure.

The admissibility decision depends only on the **candidate output**.

## Files
- `white_paper.md` — full definition (axioms, formalism, link to S, protocol, discussion)
- `axioms.md` — axioms in compact canonical form
- `tests.md` — falsifiable test suite (prompts + pass/fail criteria)
- `metadata.json` — minimal metadata (DOI, author, build timestamp)
