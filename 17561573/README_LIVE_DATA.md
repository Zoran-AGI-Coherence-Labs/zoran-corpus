# ΔBIO-COHERENCE 1001 — Données physiologiques (preuve vivante)
Ces données proviennent directement de mesures personnelles (rythme cardiaque, respiration, sommeil)
collectées via Samsung Health.  
Elles servent à démontrer le lien physique ↔ vivant dans le protocole ΔBIO-COHÉRENCE.  

Aucune donnée textuelle ni conversationnelle issue de Zoran n’est incluse.  
Tous les identifiants personnels ont été supprimés.  

Variables principales :
- time : horodatage (UTC)
- bpm : fréquence cardiaque
- brpm : rythme respiratoire
- spo2 : saturation O₂
- temp : température cutanée
- S_local : score local (β · ΔCₑ / λ)
