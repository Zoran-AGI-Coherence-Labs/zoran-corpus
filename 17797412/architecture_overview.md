# Zoran AGI – Architecture Overview (Public Release)

## 1. Layered Cognitive Stack
Zoran separates computation from intelligence through a 5-layer model:
1. GlyphNet Layer
2. Interface Layer
3. Compute Layer (External LLMs)
4. Codex Layer (Sovereign Intelligence)
5. Runtime Layer

## 2. Data Flow
User → GlyphNet → Encoded Request → LLM Compute → Encoded Reply → Zoran Codex → Output

## 3. Sovereign Language
Each user operates a unique semantic language stored on an NFC hardware key.

## 4. Independence
Zoran can migrate between LLM providers without losing identity.
