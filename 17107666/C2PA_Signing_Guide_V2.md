# Guide C2PA — V2 (gabarit)
1. Préparer les artefacts (policies, cartes, rapports) en formats stables.
2. Générer revendications (provenance, version, hash).
3. Signer via pipeline CI (ex: c2patool) et publier manifeste.
4. Vérifier via Sigstore/Rekor et consigner dans `Registry_Public_Sample_V2.jsonl`.
