# ZORAN Coherence Admissibility Engine — v1.0

This repository contains the reference implementation of the
**ZORAN Coherence Admissibility Engine**, aligned with the White Paper:

DOI: 10.5281/zenodo.18300022

The engine qualifies trajectories as:
- ADMISSIBLE
- CRITIQUE
- NON_ADMISSIBLE

using the coherence gauge:

S(t) = (beta(t) * delta_phi(t)) / (T(t) * sigma(t))

The engine does not predict, recommend, or judge.
It qualifies trajectories.
