# ZORAN Coherence Engine

An explicit, falsifiable engine to evaluate whether a claim *holds* under
formal criteria, stress tests, and invariance checks.

## Install (local)
```bash
pip install -e .
```

## Core ideas
- Define a metric
- Stress test it
- Apply falsification
- Check invariance

## Education
Coherence is treated as a *first-class subject* prior to disciplines.
