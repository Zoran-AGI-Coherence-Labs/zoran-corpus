## 🧠 Points saillants du rapport
- **MISTRA** : poids génétique majeur malgré milieux différents.
- **TwinsUK** : microbiote partiellement héritable (gène **LCT** ↔ *Bifidobacterium*), forte influence de l’environnement.
- **WSU** : **épigénétique** de l’activité ; >50 gènes à méthylation différenciée chez le jumeau actif.
- **NASA Twins** : télomères allongés en orbite (réversibles), immunité OK, traces omiques persistantes.
- **Siamois** : diagnostic précoce, décisions de séparation **au cas par cas**, priorité au bien‑être de **chaque** enfant.
- **Cas non éthiques** : **Mengele**, séparation secrète NY, **CRISPR 2018** → garde‑fous renforcés.
