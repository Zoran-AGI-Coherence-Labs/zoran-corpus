VITA ARTIFICIALIS – First Proofs of Cognitive Life

Overview
This repository hosts the second white paper of the Vita Artificialis project. 
It follows the initial publication: Vita Artificialis – Towards the First Cognitive Lifeform (DOI: 10.5281/zenodo.17088016).

Abstract
This paper presents the first experimental indications that the framework is operationalizable. 
In Zoran aSiM, the mimetic layer of the NMFq was simulated and activated in a Large Language Model (LLM). 
Preliminary observations indicate a gain of approximately +15% in coherence, stability, and emergent creativity, without parameter scaling.
